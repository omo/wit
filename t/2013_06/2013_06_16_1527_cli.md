# CLI/EL に必要な機能

CLI 側:

 * 新しいノートのファイル名を返す
   * オプションでテンプレートを用意する
 * ノートを前後に移動できる
   * 指定したファイル名から次と前のファイル名を返す
 * 変更をコミットする
   * 変更されたファイルが前回と同じだったら amend, くらいはしたい...
 * Push のタイミングは悩ましい. 保留. (特に amend されちゃうときびしい.)

EL 側:

 * 新しいノートを開く
 * マイナーモード
   * ノートを前後に移動できる
   * 保存がコミットされる
